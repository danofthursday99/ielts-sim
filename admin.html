☁️ Google Drive Config
Upload your config to Google Drive so all candidates auto-load the latest questions, audio, and passages — no matter what device they use.
Step 1 — Save & Upload Config
1. Click 💾 Save Config (top right) — downloads ielts_config.json
2. Go to drive.google.com → Upload the file
3. Right-click the file → Share → Anyone with the link → Viewer
4. Click Copy link — paste it below
Step 2 — Active Config URL
This is the URL that simulation.html and profile.html will fetch the config from. Copy this and set it in those files if needed, or it is saved automatically.
No Drive URL saved yet.
How It Works
Without Drive: Config stays in browser localStorage — only works on the admin's computer.
With Drive: Config is fetched fresh from Google Drive every time a candidate opens the test — works on any device, any browser.

⚠️ Important: Every time you change questions, audio URLs, or passages — re-save the config and re-upload to Google Drive (replace the old file). The URL stays the same.
